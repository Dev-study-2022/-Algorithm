## ✅ How To
> 📅 2022.02.07 `예정`
1. 매주 3-4문제
2. 매주 토요일 `11:00` 코드 리뷰
3. **각 주차별 Directory Name** : 'Week{num}'
    - num : 2자리 숫자로
    - ex) Week01
4. **Directory Name** : `문제번호`
    - 백준 : 2573
    - 프로그래머스 : 문제명
5. **File Name** : `문제번호_이름`  
    - 백준 : 2573_jh.java  
    - 프로그래머스 : 문제명_jh.java
6. **Commit Message** : `[Solve] ✍ 사이트명_문제번호.문제명`
